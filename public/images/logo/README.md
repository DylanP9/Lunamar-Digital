# Logo assets

Drop the real Lunamar Digital logo files here, for example:

- `lunamar-logo.svg` — full wordmark + eclipse mark (preferred, scalable)
- `lunamar-mark.svg` — icon-only mark
- `lunamar-logo-white.png` — fallback raster

Once added, update `src/components/Logo.tsx` to render the asset with
`next/image` instead of the inline SVG placeholder.
